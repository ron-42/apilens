# Welcome to API Lens

This section is for product and engineering teams using API Lens day to day.

If you are integrating SDKs or ingest APIs directly, use the **Developer** tab.

## What you will do here

1. Sign in and create your first app
2. Complete setup for your framework
3. Send traffic and confirm endpoint data
4. Filter and inspect endpoint and request data
5. Troubleshoot common issues quickly

## Before you start

- API Lens app URL: `https://app.apilens.ai`
- API Lens API URL: `https://api.apilens.ai/api/v1`

Continue with [Sign In and Access](/customer/sign-in-and-access).
